# htmlcss-discord-mobile

## Obiettivo dell'Esercizio

Riprodurre il layout mobile della pagina download di Discord focalizzandosi sulle proprietà di Position (fixed, relative, absolute) e CSS Transforms (translate).

## 🎯 Obiettivi Tecnici

* Uso di position (absolute, relative).
* Centratura e spostamenti con transform: translate.

## Tecnologie

* HTML5
* CSS3
